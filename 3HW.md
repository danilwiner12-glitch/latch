# latch
**Метастабильное состояние - это состояние цифрового устройства, при котором его выход не определён как логический 0 или 1 в течение некоторого времени. Обычно это возникает, когда входной сигнал изменяется в неподходящий момент времени (например, близко к фронту тактового сигнала).

Где возникает:
 1. триггеры (flip-flops)
 2. регистры
 3. синхронизация сигналов между разными тактовыми доменами

Методы борьбы
1. Двухтактный синхронизатор (самый распространённый)

Последовательно ставят 2 триггера:
вход -> FF1 -> FF2 -> логика

FF1 может уйти в метастабильность
FF2 получает уже «успокоенный» сигнал

2. Увеличение времени на стабилизацию
уменьшение частоты
добавление задержек

3. Специальные схемы
CDC-синхронизаторы
асинхронные FIFO
