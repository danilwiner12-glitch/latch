# latch
**Latch (защелка) - это простейший элемент памяти, который хранит 1 бит информации. Его ключевое свойство — прозрачность. Пока управляющий сигнал (например, Clock или Enable) активен, выход Q повторяет вход D (как проводник). Как только сигнал становится неактивным, latch «закрывается» и сохраняет последнее значение.**

**latch  означает что на выходе осаюися те данные которые там были написаны изначально**

E — Enable (разрешение)

S — Set (установка)

R — Reset (сброс)


*Пример самого простого кода на SV без E*
```systemverilog
if (s && !r) begin
    q = 1'b1;    
end else if (!s && r) begin
    q = 1'b0;         
end
