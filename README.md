# latch
**SR-защелка - базовый элемент памяти в цифровой схемотехнике, используемый для хранения одного бита информации. Имеет два входа: S (Set - установка, переводит прямой выход Q в 1) и R (Reset — сброс, переводит Q в 0). Два выхода: прямой Q и инверсный ~Q (всегда противоположны в нормальном режиме). Состояние запоминается и сохраняется неизменным, когда оба входа неактивны (S=0, R=0). Комбинация S=1 и R=1 является запрещенной, так как нарушает инверсность выходов и приводит к неопределенному состоянию при снятии сигналов.**

**latch  означает что на выходе остаюися те данные которые там были написаны изначально**

E — Enable (разрешение) Позволяет данным свободно проходить через защелку в активном уровне

S — Set (установка)

R — Reset (сброс)
